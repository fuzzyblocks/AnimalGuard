AnimalGuard
===========

An animal protection plugin for use in conjunction with WorldGuard.

Compilation
-----------

We use maven to handle our dependencies.

* Install [Maven 3](http://maven.apache.org/download.html)
* Check out this repo and: `mvn clean package`
